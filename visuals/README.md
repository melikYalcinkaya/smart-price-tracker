# Görseller (visuals/)

Bu klasör analiz sürecinde üretilen grafik ve görselleştirme dosyalarını içerir.

- Notebook'lardan export edilen PNG/SVG grafikler buraya kaydedilir.
- Final raporda kullanılacak görseller bu klasörden referans alınır.
- Dosya adlarında notebook adı veya tarih prefiksi kullanılması önerilir (örn. `01_fiyat_dagilimi.png`).
